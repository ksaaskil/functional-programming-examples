# Hello Haskell

Example project created with

```bash
stack new hello-haskell new-template
```

### Build

```bash
stack build
```

### Interactive mode

```bash
stack ghci
```

Load a module in `ghci`:

````ghci
Prelude> :l StartingOut
```

### Execute

```bash
stack exec hello-haskell-exe
```

### Test

```bash
stack test
```
````
